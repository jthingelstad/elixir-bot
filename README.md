## Setup

## Features
- **#leader-lounge**: @mention Elixir to ask about promotions, inactive members, roster, stats, war
- **#general-commentary**: Auto-posts at 7am, 12pm, 5pm, 9pm (America/Chicago)

## Commands (mention in #leader-lounge)
- `@Elixir who should we promote?`
- `@Elixir who is inactive?`
- `@Elixir should we kick [name]?`
- `@Elixir clan stats`
- `@Elixir roster`
- `@Elixir war status`
